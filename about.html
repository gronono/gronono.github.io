À Propos

Ce site est le blog personnel d’Arnaud Brunet.

Hébergement

Le site est hébergé gracieusement par la société GitHub via le système des Github Pages.

Le nom de domaine gronono.fr est enregistré chez la société OVH.

Moteur de blog

Le site utilise le moteur de blog Jekyll.

Le design du site est très largement inspiré d’EasyBook dévelopé par laobubu.

Système de commentaires

Le site utilise le système de commentaires fourni par Disqus. Pour commenter vous devez utiliser un compte Disqus, Facebook ou Twitter ou tout autre fournisseur d’authentification proposé par Disqus. Ces derniers peuvent recueillir des données personnelles lorsque vous naviguez sur ce site sans que l’auteur ne soit au courant.

Lorsque vous postez un commentaire vous le faites sous votre responsabilité.

Outils tiers

Ce site utilise des outils fournis par des tiers. En naviguant sur ce site, ces derniers peuvent récolter des données vous concernant sans que vous le remarquiez.

Liste des tiers :

  • Cloudflare : fournisseur de CDN
  • Fontawesome : fournisseur d’icônes
  • MathJax : moteur de rendu de formules mathématiques
